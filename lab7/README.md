## Лабораторна робота №7
Детекція облич за допомогою нейронної мережі (ResNet-10) з використанням багатопотоковості.

### Архітектура
- `FaceDetector`: Клас, що працює у фоновому потоці (Worker Thread), виконує `net.forward()`.
- `main`: Основний потік (UI Thread), зчитує кадри, відображає результат та обробляє клавіатурні події.
- Синхронізація: Використано `std::mutex` та `std::condition_variable`.

### Як запустити
1. Перейдіть до папки лабораторної: `cd lab7`
2. Надайте права на виконання: `chmod +x *.sh`
3. Встановіть залежності: `./preinstall.sh`
4. Зберіть проєкт: `./build.sh`
5. Запустіть: `./run.sh`
