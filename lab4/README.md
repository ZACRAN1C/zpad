# Лабораторна робота №4: Візуалізація даних та інтерактивні інтерфейси

## Опис завдання
Реалізація програми для генерації та інтерактивного відображення гармонічного сигналу виду $y(t) = A \cdot \sin(\omega t + \phi)$ із накладеним шумом. Програма містить цифрову фільтрацію зашумленого сигналу за допомогою фільтра Баттерворта (SciPy) та надає користувачу графічний інтерфейс (Matplotlib) для керування параметрами в реальному часі.

## Вимоги до системи
* **Операційна система:** Windows 10/11, macOS або Linux
* **Інтерпретатор:** Python версії 3.10 або новішої (тестовано на Python 3.13)
