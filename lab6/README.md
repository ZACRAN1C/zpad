# Лабораторна робота №6: Обробка відеопотоку за допомогою OpenCV

### Опис завдання
Програма реалізує модульну архітектуру для захоплення відео з камери та обробки зображення в реальному часі у режимах: Original, Invert, Gaussian Blur, Canny.

### Вимоги до системи
* **ОС:** Ubuntu 22.04+ / Linux
* **Компілятор:** GCC 11+ / Clang
* **Збірка:** CMake 3.20+

### Інструкція із запуску (Linux)
Для автоматичного розгортання та запуску використовуйте скрипти:
1. Завантаження пакетів: `./preinstall.sh`
2. Компіляція проєкту: `./build.sh`
3. Запуск бінарника: `./run.sh`
