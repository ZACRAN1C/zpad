# Лабораторна робота №5: Наука про дані: обмін результатами та початковий аналіз


---

##  Опис завдання
Метою роботи є створення інтерактивного веб-додатка за допомогою бібліотеки `Streamlit` для візуалізації та аналізу екологічних індексів (VCI, TCI, VHI) на основі історичних даних NOAA, отриманих під час виконання Лабораторної роботи №2.

### Функціонал додатка:
- Інтерактивні випадаючі списки для вибору екологічного індексу та області України.
- Слайдери для фільтрації даних за діапазоном тижнів та років.
- Кнопка швидкого скидання всіх встановлених фільтрів до початкового стану.
- Розділення інтерфейсу на дві робочі колонки (контролери ліворуч, аналітика праворуч).
- Чекбокси сортування даних за зростанням/спаданням з автоматичним вирішенням колізій.
- Три вкладки відображення даних: таблиця результатів, лінійний графік часового ряду, графік порівняльної динаміки обраної області з іншими регіонами.
